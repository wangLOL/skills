# Deep Stock Analysis 深度股票分析框架

## ⚠️ 硬性规则

1. **强制使用 Tavily 搜索** - 所有数据必须通过 Tavily API 获取，禁止使用其他搜索方式

## 交易日数据规则（强制）

| 当前北京时间 | 使用数据 |
|-------------|---------|
| < 15:00 | 前一交易日收盘 |
| ≥ 15:00 | 当日收盘 |

**PE/PEG计算**：
- PE (TTM) = 最新收盘价 ÷ EPS(TTM)
- PEG = PE ÷ 净利润增速(%)

## 标准报告结构

### 一、投资摘要

| 项目 | 数据 | 来源 |
|------|------|------|
| 当前股价 | XX元 | X月X日收盘（Tavily） |
| 市值 | XX亿元 | 基于收盘价计算 |
| 净利润 | XX亿元 | 同比+XX% |
| PE (TTM) | XX倍 | 基于收盘价计算 |
| PEG | XX | PE÷增速 |
| 机构评级 | 买入/观望 | - |
| 目标价 | XX-XX元 | 券商研报 |

### 二、业务结构分析
- 收入结构（分产品/分地区）
- 毛利率贡献
- 客户集中度（前五大客户占比）

### 三、订单与增速分析
- 核心客户订单情况
- 订单增速（同比、环比）
- 在手订单/合同负债

### 四、上游行业分析
- 上游原材料供给
- 上游价格走势
- 成本影响

### 五、下游行业分析
- 下游需求端客户
- 市场规模
- 驱动因素

### 六、公司最近动态（1-3个月）
- 股价波动
- 公司公告
- 高管增减持

### 七、竞争格局
- 市场份额
- 竞争壁垒
- 竞品对比（至少2家）

### 八、成长逻辑验证
- 利润增长来源（真成长/降本/并表）
- 增长质量（内生vs外延）
- 持续性判断（行业β + 公司α）

### 九、估值分析
- 相对估值（PE/PB/PS）
- 敏感性分析（不同增速假设下的PE/PEG）
- 目标价区间

### 十、二级市场参与度
- 融资融券余额
- 主力资金流向（近5日/近20日）
- 龙虎榜（如有）
- 股东结构变化
- 舆论情绪

### 十一、风险清单

按严重程度分级：
- 🔴 **高风险**（直接影响商业模式）
- 🟡 **中风险**（影响短期业绩）
- 🟢 **低风险**（可接受）

每类至少列出1-3条。

### 十二、投资建议

- **综合评级**：⭐ 1-5星
- **买入条件**：明确触发条件
- **止损位**：具体价格
- **目标位**：乐观/中性/悲观三种情景
- **观察指标**：需要跟踪的关键指标

1. **表格优先** - 能用表格的不用文字
2. **数据标注** - 每个数据点后标注来源（如：Tavily 2025Q3财报）
